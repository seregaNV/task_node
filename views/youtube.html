API YouTube


To get started working with youtube api, enter the required data:

API Key
This field is required for input. You can get an key in https://console.developers.google.com'.
Max results
In this field you can specify the quantity of the search results. If this field is empty, the quantity of search results will be equal "5".

From the three fields, you must fill out only one.

Keywords
In this field you can enter keywords to search.
Video ID
In this field, you need to enter ID of video.
Playlist ID
In this field, you need to enter ID of playlist.
